### Intellij

ctrl + alt + m     抽取方法
ctrl+o   查看可以重写的父类方法
Ctrl+alt+o 清除多余的包
Ctrl+Alt+L，格式化代码
Ctrl+q 查看帮助文档
Shift + f6  整体重命名变量值


alt+ shift + up/down 向上或向下移动一行代码。



ctrl +w   选中光标所在的单词 ，连续按就是选中连续的单词，单词选完以后，再按就是选中整体方法





### AndroidStudio

Toast + tab  自动补全toast

Ctrl+alt+T try catch 等等

log +t  构建一个带有tag 的public static final 的字符串。

log  + d 输出



alt + 上箭头 或者下箭头  方法之间移动

ctrl + alt  左右箭头  追踪源码，退回源码
